Como Usar o UniLab

Siga estes passos simples para começar a usar o sistema

1

Escolha seu perfil de acesso

Selecione entre Administrador ou Professor para acessar as funcionalidades específicas para seu perfil.

2

Faça login no sistema

Utilize suas credenciais institucionais para acessar o sistema de forma segura.

3

Navegue pelo painel de controle

Explore as diferentes seções do sistema para gerenciar laboratórios, agendamentos e relatórios.

4

Realize agendamentos

Selecione o laboratório, data e horário desejados para criar um novo agendamento.

5

Visualize e edite agendamentos

Acompanhe os horários reservados na área de agendamentos e utilize a função "Editar" para fazer ajustes quando necessário.
